Content creation guidelines

In this webpage we will go over everything you need step by step to create content for WWCP We will cover the following big sections

  1. Determining the size
  2. General Requirements
  3. Not allowed details
  4. Texture Requirements

1. Determining the size

In content creation, the first big step is, of course, determining the size of the rolling stock piece you
are designing. As you all know WWCP uses different scaling methods for the following big regions:

  • European
  • North America
  • UK

Under this, you will find the various requirements regarding size and the calculators you should use. But before we go to that I just want you to know the general width used in the different regions.

  • European: 22mb wide body. 24mb with exterior details
  • North America: 22mb wide body. 24mb with exterior details
  • UK: 20-22mb wide body. 22-24 with exterior details
  • European Content:

    For EU content the general rule of thumb is using the standardized European buffer height which is set in SMP/FMT at Y = 1 for the following piece and then scaling your technical diagram in accordance. Nonetheless, we advise you to use the calculator below to double-check height and length. The standardized European buffer is a default requirement to be placed on your model. Exceptions can be requested for heavily differentiating stock.


  • UK Content:

    For UK content the standardized UK buffer height which is set in SMP/FMT at Y = 0.5 can be used for scaling your technical diagram in accordance. Nonetheless, we advise you to use the calculator below to double-check height and length. The standardized UK buffer is a default requirement to be placed on your model. Exceptions can be requested for heavily differentiating stock.


  • North American Content:

    For American content, the general rule of thumb is using the standardized American knuckler height which is set in SMP/FMT at Y = 3.5 for the following piece and then scaling your technical diagram in accordance. Nonetheless, we advise you to use the calculator below to double-check height and length. The standardized knuckle is a default requirement to be placed on your model. Exceptions can be requested for heavily differentiating stock.

2. Actual Requirements

In this section, we will be talking about general requirements for the big outlines on content and on minima and maximum detail requirements. We will also shine a light on any texture requirements.

General Requirements

  • You must follow the Length and Height Calculators provided above.
  • The wheels should be placed 12mb’s apart of each other.
  • In SMP/FMT the arrow direction marks the front of the Locomotive/Wagon
  • The floor in SMP/FMT should be interpreted as the top of the track level
  • Roofs may not consist of more than 9 pieces.
  • Circles must be done using an Octagonal shape>
  • We do not accept any “Shape’s” only Shapeboxes or regular boxes.
  • Doors must be separate geometry and custom-named for example “front door left”.
  • Lamps, lamp sockets, headlights, ditch lights, etc must be separate geometry and custom-named for example “Front Lamp”
  • Windows should generally be textured on. We do make a case per case exception if the windows have to be placed on HEAVILY curved, sloped or Streamlined window designs. If a real-life model has a literal hole in its frame you are required to model the hole of course.
  • Fluting on sides is allowed to be small geometry strips of minimum 0.1 wide. Fluting on roofs should be textured on.
  • Numberboards must be a separate piece of geometry
  • Railings should always be square. For example, if it is 0.25 wide it should be 0.25 high. Exceptions can be requested if the real-life equivalent has odd proportions on its railings.
  • Try to do as much as you can with texture and use as little geometry as possible while still following the below mentioned extra requirements.
  • Wheels should have their rotation points on the axle.
  • All parts must be in the “Core” group of SMP. (not applicable for FMT models)
  • If your model will include a 20ft, 40ft container or any other type of container you need to use the one provided by WWCP.
  • The top of a pantograph should be at a maximum of 40mb’s higher than the bottom of the wheels.
  • Bogie’s need to be a separate model
  • If a freight car has a load this must be modelled.

Minimum Detail Requirements

  • The model must represent the real-life version as accurate as possible following the guidelines.
  • A bare minimum locomotive drive stand has to be included.
  • Basic seating must at least be provided where necessary.
  • Big boxes on the undercarriage if the real-life variant has this too.
  • Bogie attachment points.
  • Ladders/railings in corresponding places.
  • Valve gears/rods if applicable
  • walkways on passenger cars if applicable

Maximum allowed detail Level

  • The undercarriage must be flat with the only exception being highly visible equipment that is mounted under the frame and bogie attachment points.
  • Horns, Vents, bells, Electrical wires on top of the locomotive are all allowed to be geometry.
  • Text signs are allowed to be up to 4 times the normal texture resolution. Achieve this with rescaling shape boxes. (Exceptions can be requested)
  • Logo signs are allowed to be made using rescaled shape boxes. The logo on the texture net may not take up more than 160px’s
  • Trains are allowed to have a walkway if applicable.
  • A basic cab design with a maximum of 20 gauges, meters, levers, valves or screens.

Special Requirements for Streamlining:

  • No more than 7 slopes per axis, per side, not including 0 or 90-degree geometry
  • Keep it minimal, without compromising the general look of the form
  • No folded geometry (split the part into two triangles if it is folded)
  • Reduce texture stretching as much as possible, especially where lining or other visually different textures will be applied
  • If streamlined cabs can not be modelled in a way that the texture of the windows is correct, modelled windows are acceptable but must be made to look textured.

3. NOT ALLOWED DETAILS

  • Detailed underframes (Litteral framework)
  • Gas/Water/Fuel/Sand caps unless this is bigger than 1.5mb on any axis.
  • Boiler piping
  • Full Blown detailed, Realistic cab interior
  • Unlike horizontal fluting, Vertical wooden structures need to be textured on instead of geometry

4. Texture Requirements

  • We require all visible parts on a certain skin to be textured fully and no SMP/FMT default colours
  • We require colours to be between V:12 and V:88
  • For noising skins, we use the following settings: 10-12, 0, 50
  • Skins are allowed to have company logos on them. But are in no way absolutely required to.
